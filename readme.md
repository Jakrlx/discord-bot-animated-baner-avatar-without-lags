# Animated avatar and banner for a Discord Bot - Without input lags

This is a simple script that allows you to change your bot's avatar/banner to an animated one.

## How to use?

1. Clone the git repo or download it in .zip format

```js
git clone https://github.com/FaeWulf/animated-bot-icon.git
```

2. Install dependencies (npm, pnpm,...) NOTE! You can skip this if the "node-modules" folder already exists.

```js
npm install
```

3. Place your bot token in the .env file

```js
token="PASTE THE BOT TOKEN FROM THE DEV PORTAL HERE"
```

4. Place the icon/banner in the repo folder and rename the icon -> logo.gif, the banner -> banner.gif

5. Run the script

```js
node index.js
```

NOTE!: The .GIF file size must be less than 10 MB
